== At a Glance

This is a web app to generate trivia quiz, specifically built for a University 
tryouts. It allows students who took the tryout to share their result to their 
parent, and it has Scholarships to be applied on a certain amount of score on
tryouts.

== Features

* Quiz with JSON data type
* Scholarships determination
* Tryout result sharing
* Administration (uses ActiveAdmin)

== Requirements

* Unix-like systems
* Ruby 2.1.2
* PostgreSQL 9.3.x
